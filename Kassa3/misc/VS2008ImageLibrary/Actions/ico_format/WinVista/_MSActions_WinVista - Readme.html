Библиотека рисунков Visual Studio: действия (WinVista)

Рисунки значков
Этот тип файла может содержать более одного размера изображения и/или количества используемых цветов. Эти файлы обычно используются в элементах управления, которые предусматривают варианты стиля представления; примером является список, позволяющий пользователю выбрать между "просмотром мелких значков" и "просмотром крупных значков". Элемент управления отображается с использованием соответствующего размера изображения и количества цветов. Предварительный просмотр всех файлов значков наилучшим образом возможен с помощью Проводника Windows; при просмотре в обозревателе отображаются не все варианты, имеющиеся в файле.

Ограничения использования
Следующие изображения (или любая часть этих изображений) должны использоваться как часть визуального языка в определенном соответствии с приведенным ниже описанием:

Имя файла Имя файла Использование
autolist.ico автоматическое создание списка элементов
autoplay.ico автоматическое воспроизведение
autorun.ico автоматический запуск программы
burnCD.ico запись на компакт-диск
connect_toNetwork.ico подключение к сети
copy.ico копирование в буфер обмена
cut.ico удаление в буфер обмена
delete.ico удаление файла без возможности восстановления
move.ico перемещение выбранного элемента
paste.ico вставка из буфера обмена
personalization.ico настройка интерфейса
rename.ico переименование выбранного элемента
replace.ico замена выбранного элемента
search.ico поиск
