Библиотека рисунков Visual Studio: макрокоманды (Office & Dev)

Иллюстрации значков
Этот тип файла может содержать более одного размера изображения и/или количества цветов. Эти файлы обычно используются в элементах управления, которые предлагают варианты стиля представления; примером является список, позволяющий пользователю выбрать между вариантами "представление маленького значка" и "представление крупного значка". Элемент управления отображается с использованием соответствующего размера изображения и количества цветов. Для всех файлов значков наилучшим способом предварительного просмотра является использование приложения Windows File Explorer, так как в обозревателе не отображаются все варианты, имеющиеся для файла.

Ограничения применения
В качестве части визуального языка следующие изображения (или любая часть изображений) должны использоваться согласованно с указанным ниже описанием:

Имя файла Имя файла Применение
servicepaused.ico работа службы приостановлена
servicerunning.ico служба запущена
servicestopped.ico работа службы остановлена
serviceunknown.ico     состояние службы неопределенно
